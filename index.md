# プライバシーポリシー

MeditationJourney（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本ポリシーでは、本アプリにおける情報の取り扱いについて説明します。

## 1. 取得する情報
- ユーザーが本アプリ内で記録した瞑想時間・利用履歴（端末内保存）
- ユーザーが選択した国・地域や再生したサウンドに関する情報（端末内保存）
- 60分以上瞑想した場所を示すマップピン情報（端末内保存）

氏名、住所、電話番号、メールアドレス等の個人を特定できる情報は取得しません。

## 2. 情報の利用目的
- 瞑想記録の保存・表示
- カレンダー表示やマップピン表示など、アプリ機能の提供
- 利用状況に応じた体験向上のための表示・管理

## 3. 第三者提供
本アプリは、法令に基づく場合を除き、取得した情報を第三者へ提供しません。

## 4. 外部サービスの利用
本アプリは、広告配信サービスおよび外部解析サービスを利用していません。
